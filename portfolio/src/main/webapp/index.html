Research Assistant at Georgia Tech’s Augmented Reality Experiences Lab

I am interested in animation, augmented and virtual reality, so I became a research assistant at the Georgia Tech Augmented Reality Lab. I worked on creating models of my face in Blender. This included box modeling and face carving, which gave my model a similar face structure as mine, and adding topography similar to my real face's on my model, which made movement of my model realistic.

Co-Founder

In my first year at my university, I was a member of the Grand Challenges Living Learning Community, a community dedicated to creative and innovative thinking. I decided to work on education since that is something I am passionate about.
In summary, I and four others found that a good number of adults were not financially literate. We performed some behavior analysis on Georgia Tech college students which proved our hypothesis.

Sudoku Solver

I am a huge fan of sudoku (I have a collection of sudoku books, all solved), and I had some extra time on my hands so I decided to challenge myself. I implemented a sudoku solver that would solve easy level sudoku puzzles.
I made use of recursion mostly with this program, checking till there was no blank spaces left. Initially, I made use of Java only, having the user to input a 2-dimensional array of integers and then returning the result in the same fashion. I am currently working on recreating this program using ReactJS so as to make it more user friendly.

Collaborative Code Editor

I worked on this project with a couple of my teammates from my Software Product Sprint team. We created a code editor that allowed users write code in a selection of languages.
Our editor is able to highlight keywords and variables in various colors in order to differentiate them easily. It also allows our users save and retrieve their code without refreshing the page. Additionally, as the name implies, our editor allows multiple users work on the same piece of code. So yes, you may use our editor for your technical interviews (jk haha).
To work on this project, I learnt and made use of HTML, Javascript, CSS, and Ajax.
